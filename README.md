## ☁️ Weather Forecast Login - WeatherMB

A web application developed to provide real-time weather information with a clean and modern interface.

## Technologies
* HTML5
* CSS3
* JavaScript (API Integration)

## How to run the project
1. Clone this repository.
2. Open `index.html` using the **Live Server** extension in VS Code.
3. Enter a city name in the search field to get the current weather data.
